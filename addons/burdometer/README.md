# Burdometer #

* Monitors automaton burden values.
* Port of windower v4 Autocontrol's burden tracking feature.

----

**Main Command:** `/burden`

#### Commands: ####
* 1: pos [x] [y] - Set burden window position.
* 2: style [window_style] - Set burden window style.
